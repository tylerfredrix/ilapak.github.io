Changing the wrap

Splicing Technique:

  • Loosen wrap and pull off enough to splice to new roll using red tape.

  • Using the electric lifter put a new roll of wrap on.

  • Align the black tabs on roll and stick tape.

  • Tighten roll and press the tension arm to tighten wrap.
    Be sure to keep wrap on even.

  • Start machine back up and watch for the piece with the splice. Remove the piece with splice and the piece after.
    Also keep an eye on the code date to make sure it is in correct spot.
