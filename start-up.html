Start up

Follow these steps for start up once the machine has been assembled and dry.


  1. Check the product schedule to find which product will be ran. Install the proper wrap accordingly.

  2. Load the proper program for cheese type.

  3. Verify the proper tamper pads are installed for product type.

  4. Make adjustments for tamper height and missplace sensor according to the product info page on ilapak control screen.

  5. Set the code dater to the proper date. Verify it at the inkjet dater located at the taper outlet.
